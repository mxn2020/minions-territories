---
title: Installation
description: Install Minions Territories packages
---

## TypeScript / Node.js

```bash
npm install @minions-territories/sdk minions-sdk
```

## Python

```bash
pip install minions-territories
```

## CLI

```bash
npm install -g @minions-territories/cli
```

## Requirements

- **Node.js** 20 or later
- **Python** 3.11 or later (for Python SDK)

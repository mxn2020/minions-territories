# Minions Territories

Country, state, region, city definitions, focus queue, and saturation tracking

## Overview

This project is part of the [Minions ecosystem](https://github.com/mxn2020/minions) and provides:

- **TypeScript SDK** (`@minions-territories/sdk`) — Core library for Node.js
- **Python SDK** (`minions-territories`) — Python bindings
- **CLI Tool** (`@minions-territories/cli`) — Command-line interface
- **Web App** — Interactive playground at [territories.minions.wtf](https://territories.minions.wtf)

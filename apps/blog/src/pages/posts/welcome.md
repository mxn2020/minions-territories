# Welcome to Minions Territories

We're excited to introduce **Minions Territories** — Country, state, region, city definitions, focus queue, and saturation tracking.

## What's Included

- **TypeScript SDK**: `npm install @minions-territories/sdk`
- **Python SDK**: `pip install minions-territories`
- **CLI**: `npm install -g @minions-territories/cli`

## Getting Started

Check out the [documentation](https://territories.minions.help) for a full guide.
